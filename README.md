# Copper Bulb Ealy
这是一个将 ``1.21.4 铜灯 (Copper Lamp)`` 完美移植到 ``1.20.1`` 的模组，将铜灯完整特性与成就带回 1.20.1。  
因为mod使用了原版名称空间和数据格式，所以可以与Litematica Mod读取的高版本投影内的铜灯兼容，同时升级存档也能被高版本识别并保留。  
  
## 环境要求
``Minecraft-1.20.1`` ``Fabric-0.16.10+``  
``多人服务器``：**服务端与客户端均必须安装**本模组。  
``单人游戏``：**仅客户端**安装即可。  
  
**注意**：
如果仅在服务器安装而客户端未安装，连接服务器时会出现“模组缺失”或方块 ID 无法解析的问题。  
如果仅在客户端安装，则可以正常进入无此Mod的服务器（Mod方块不生效）。
